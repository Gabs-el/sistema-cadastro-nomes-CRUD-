# 🧾 Sistema de Cadastro de Nomes (Python)

## 📌 Descrição

Aplicação em Python via terminal que simula um sistema simples de gerenciamento de dados (CRUD), permitindo cadastrar, listar, buscar e remover nomes.

## ⚙️ Funcionalidades

* ➕ Cadastrar nomes (com validação para evitar duplicados)
* 📋 Listar todos os nomes cadastrados
* 🔍 Buscar nomes na base de dados
* ❌ Remover nomes com confirmação
* ⚠️ Tratamento de erros de entrada

## 🧠 Conceitos aplicados

* Estruturas de repetição (`while`)
* Condicionais (`if`, `elif`, `else`)
* Listas e manipulação de dados
* Funções para organização de código
* Tratamento de exceções (`try/except`)
* Boas práticas de entrada de dados (`strip`, `lower`)

## ▶️ Como executar

```bash
python main.py
```


Projeto desenvolvido com foco em fortalecer lógica de programação e manipulação de dados em Python, simulando um sistema real em ambiente de terminal.
